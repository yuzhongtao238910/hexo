关于取消请求和react native的环境配置的两个知识点

取消请求

目前普遍使用的是axios库，其包含了一个属性cancelToken,在0.22版本之前是可以使用该属性进行取消请求的

但是从0.22版本之后改为了使用controlAbort,该方法在mdn上面有详细的解释，这里不再过多的说明

react native的环境配置

首先这个环境是真的不好配置，目前我只配置的ios的开发环境，坑爹的要死，这里只说两点
1- mac内置的ruby版本的2的版本，需要使用最新的3版本才可以
2- 在下载homebrew的时候会各种卡顿，大家可以搜索中科大源或者清华源进行配置
3- 在react native初始化之后，需要进入ios包并且使用pod install命令进行安装cocoapods包
