关于js之中的数据类型的转换的原则

原始 -》 数字 调用Number()
  • true 1
  • false 0
  • null 0
  • undefined NaN
  • NaN NaN
  • string
    • 空字符串（含空白字符\t\r\n）：0
    • 去掉引号：不是数字就是NaN
    • 存在空格的话，会去掉首尾的空格来转换
所有 -》 bool 调用Boolean()
  • 0 false
  • undefined false
  • null false
  • NaN false
  • 空字符串 false 只有空字符串是false
  • 其他都是true
原始 -》 字符串 调用String()
  • null “null”
  • NaN “NaN “
  • undefined “undefined”
  • number “数字”
  • true “true”
  • false “false”

对象 -》 原始

  • 调用valueOf 得到的是对象的话，就会调用toString()，如果toString得到的还是对象的话，就会报错
