三个新发现的api/或者能够称为面试题吗？

1- symbol

symbol 是一种基本数据类型;每个从 Symbol() 返回的 symbol 值都是唯一的。一个 symbol 值能作为对象属性的标识符

但是：Symbol.for(key) 方法会根据给定的键 key，来从运行时的 symbol 注册表中找到对应的 symbol，如果找到了，则返回它，否则，新建一个与该键关联的 symbol，并放入全局 symbol 注册表中。

2- flex 的 对应的属性，看到过对应的面试题
- flex-basic: CSS 属性 flex-basis 指定了 flex 元素在主轴方向上的初始大小。如果不使用 box-sizing 改变盒模型的话，那么这个属性就决定了 flex 元素的内容盒（content-box）的尺寸。
注意： 当一个元素同时被设置了 flex-basis (除值为 auto 外) 和 width (或者在 flex-direction: column 情况下设置了height) , flex-basis 具有更高的优先级。
- flex-direction: 布局方向
- flex-wrap: 是否换行
- flex-shrink: 收缩规则，flex 元素仅在默认宽度之和大于容器的时候才会发生收缩，其收缩的大小是依据 flex-shrink 的值。
- flex-grow: flex 项 主尺寸 的 flex 增长系数。 简而言之，占据几份

  • flex-flow:  属性是 flex-direction 和 flex-wrap 的简写。
